The data are divided according to the transfer conditions.

The folder name consist of "object rotation angle(object size)".

For example, CCW_4(2) denotes that rotation angle is CCW_4 and size is No.2.

There are 10 CSV files in each folder because the experiment was repeated 10 times for each condition.

CSV file name consist of "Data collection data_object size_iteration number_speed of conveyor belt"

The first column of the CSV file is the time and the second column is the measured distance.
